## Verdict: the method fixes work; the coverage gap is *not yet* closed (new cause)

The three fixes changed the agent's behaviour exactly as intended at the analysis level, but a **budget/seed-count mismatch** meant the headline PE vs EH gap still wasn't characterised. Net: clear methodological progress, headline target still open.

## What improved (fixes verified)

**1. Per-comparison ranked GSEA is now actually used.** The agent called `gsea_enrichment` on individual DEG tables and reasoned on signed NES + adj_p, instead of enrichment on hand-picked gene lists. This is the core Fix-1 behaviour and it is working.

**2. The down-regulated proliferation/OXPHOS axis is recovered.** Previously absent entirely. The new run repeatedly surfaces HALLMARK_E2F_TARGETS / G2M / MYC **DOWN** in lesions and concludes *"Healthy endometrium shows the most active proliferative program relative to OE and PE — endometriosis lesions exhibit reduced proliferation."* This matches the ground truth, where E2F/G2M/MYC/OXPHOS are DOWN in 4–5 of the 6 contrasts. Direct hit on a signal the baseline missed completely.

**3. OE vs EH biology now matches GT closely.** GSEA surfaced COMPLEMENT (C7, CFH, C3), EMT, INFLAMMATORY, TNFα **UP** and E2F **DOWN** — aligning with the GT (OE vs EH: EMT + COMPLEMENT + INTERFERON UP, proliferation DOWN). The complement signal in particular is now explicit, not just "C7 is a top gene".

**4. The false "DIE ≈ PE, >99.9% identical" claim is gone.** The baseline's H4 over-statement did not reappear — the Fix-2 guard against declaring groups near-identical held. PE's proliferation-DOWN half was even captured (EH vs PE: cell-cycle UP in EH → down in PE).

## What did NOT improve

**1. PE vs EH immunological signature still missing.** GT for PE vs EH is dominated by ALLOGRAFT_REJECTION (NES +2.20), INFLAMMATORY_RESPONSE, and MHC-class-II antigen presentation (GO fold 12–16×). The new run captured PE's *proliferation-DOWN* half but never surfaced the *immune-UP* half. Two reasons:
   - The direct PE vs EH seed (S4 = REACTOME_COLLAGEN_DEGRADATION on the single small GSE11691 table) was left **PENDING — never evaluated** (see below).
   - On the larger EH vs PE table the agent anchored on the top seed (E2F proliferation) and did not inspect the opposite-direction immune sets.

**2. NEW problem — seed inflation vs hypothesis budget.** The data-driven seeder now emits **18 seeds** (14 GSEA + 4 cross-dataset) but the run used **max_hypotheses=10**. Result: **8 seeds PENDING, 9 UNCERTAIN, 1 CONFIRMED.** Several whole comparisons (including direct PE vs EH, S4) were never evaluated. The structural-coverage intent of Fix 3 is defeated if the hypothesis budget is smaller than the number of comparisons.

**3. Power — per-single-dataset GSEA rarely reaches significance.** Only the single largest table (DEG 13, OE vs EH, 27k genes) reached adj_p<0.05 (473 sig pathways). Everywhere else NES was strong and directionally correct but adj_p ≈ 0.06–0.16. The manual GT reaches significance because it is a **meta-analysis across datasets**; the agent runs GSEA on one small table at a time and is underpowered. The agent correctly attributed this to heterogeneity/sample size.

## Recommendations (next iteration)

1. **Make the budget cover the comparisons.** Auto-set `max_hypotheses ≥ number of unique comparisons`, or have the runner evaluate **one seed per unique comparison before any duplicates**, so no comparison is left PENDING. This single change would have forced PE vs EH (S4) to be evaluated. Re-run with `max_hypotheses ≥ 14`.
2. **Meta-GSEA for power.** Let `gsea_enrichment` aggregate datasets that share a comparison (rank/NES meta-combination) so per-comparison enrichment has the power the manual fgsea meta-analysis has. Single small tables rarely clear FDR<0.05.
3. **Two-sided seeding.** When a comparison's top |NES| set is one direction, also seed/inspect the strongest opposite-direction set — otherwise PE's immune-UP signal is masked whenever proliferation dominates the top seed.

**Bottom line for the manuscript:** the fixes demonstrably (a) recovered the proliferation/OXPHOS down-axis, (b) made OE-vs-EH complement/EMT explicit, and (c) eliminated the spurious DIE≈PE identity claim — all verifiable against the manual fgsea. The remaining PE-immune gap is now a clean, well-understood budget+power issue with a concrete fix, not a reasoning failure.
